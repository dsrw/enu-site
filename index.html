Enu is for learning to code and making games.

Code with friends.

Build using code, by hand, or both.

Control complex behavior with nested state machines.

Shorthand commands for new typists.

100% free and open source.

Enu is MIT licensed and free forever, to use however you like.

It uses the Nim programming language and runs on the Godot game engine.
